# Versions

## v1-lite (Now)

**Capability focus**

- Single-workspace campaign operations for agencies (no formal client boundary yet)
- Contact/audience management with segmentation via tags + simple rule filters
- Campaign creation, template reuse, scheduling, and status tracking
- Send modes: immediate, scheduled-once, recurring (daily/weekly)
- Baseline analytics only:
  - send count
  - delivered
  - open rate
  - click rate
  - unsubscribe rate
  - bounce rate

**Timeline intent**

- Current implementation planning target
- Used to validate product usefulness before deeper platform expansion

## v2-standard (Next)

**Capability focus**

- Formal client/account boundary model for agency workflows
- Expanded segmentation logic and smarter campaign audience tooling
- Stronger campaign workflow controls and operational governance
- Initial external integration foundations

**Timeline intent**

- Starts after v1-lite validation proves workflow utility and scope stability
- Prioritizes structural upgrades before enterprise-grade feature depth

## v3-expanded (Later)

**Capability focus**

- Advanced automation workflows/journeys
- A/B testing and deeper analytics/attribution
- Collaboration primitives, approvals, and role-based permissions
- Multi-channel capabilities and broader integration ecosystem

**Timeline intent**

- Long-horizon expansion tier
- Enter only when v2-standard client model + workflow controls are stable
