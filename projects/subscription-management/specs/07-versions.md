# Versions

## v1-lite (Now)

**Capability focus**

- Solo-user subscription inventory in one place
- Status tracking (`active`, `trial`, `paused`, `canceled`)
- Cost, billing cycle, renewal visibility
- Usage signal (`in use`, `not in use`, `unknown`)
- Worth tracking (1-5 score + optional note)
- Local-first persistence (`localStorage` + `SQLite WASM`)

**Timeline intent**

- Current active versioning target
- Used for first practical validation loop and POC preparation

## v2-standard (Next)

**Capability focus**

- Household collaboration model introduction
- Cloud sync foundation over migration-safe local schema
- Stronger reminder/review workflows
- Better filtering/grouping and subscription health views

**Timeline intent**

- Planned after v1-lite proves clarity and workflow usefulness
- Prioritize only once v1-lite scope is validated, not assumed

## v3-expanded (Later)

**Capability focus**

- Smart recommendations (keep/pause/cancel signals)
- Deeper integrations (billing/payment/import sources)
- Advanced analytics, long-term trends, and optimization insights

**Timeline intent**

- Long-horizon expansion tier
- Enter only after v2-standard collaboration and sync behavior stabilizes
